# Online Shop



## Requirements

- Java 17+
- Node.js 20.x

## Installation

- Clona el repositorio:

   ```bash
   git clone https://github.com/AlexanderPumaPucho/online-shop.git
   cd online-shop
   ```

- Build the project:

   ```bash
   ./mvnw clean install
   ```

- Run the application:

   ```bash
   ./mvnw
   ```

  ```bash
  http://localhost:8080/
  ```
